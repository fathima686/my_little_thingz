Integration Guide

🔧 Where is the AI Feature?

⚠️ Important: The AI image generation is currently a separate backend service. It's NOT yet integrated into your admin "Design Editor" interface.
✅ What's Working:
  • AI Service is running at http://localhost:8001
  • Standalone test UI is available
  • API endpoints are ready
  • React component is created

Current Access Points:

1. Standalone Test Page

Open this URL in your browser:

http://localhost:8001/test-ui.html

This is a separate page where you can test AI image generation without any integration.

2. API Endpoint (For Developers)

Use PowerShell to test the API:

$body = @{ prompt = "a golden trophy" } | ConvertTo-Json
Invoke-RestMethod -Uri http://localhost:8001/generate-image -Method Post -Body $body -ContentType "application/json"

3. React Component (Not Integrated Yet)

Location: frontend/src/components/admin/AIImageGenerator.jsx

This component is created but needs to be added to your Design Editor.

To Add AI to Your Admin Interface:

Step 1: Locate Your Design Editor

Your design editor is at: frontend/admin/design-editor.html

Or in the React version: frontend/src/components/admin/TemplateEditor.jsx

Step 2: Add AI Button

Add a button in the toolbar that says "Generate AI Image" or shows a sparkle icon (✨)

Step 3: Connect to AI Service

When the button is clicked, call the AI service API:

fetch('http://localhost:8001/generate-image', { method: 'POST', body: JSON.stringify({ prompt: userPrompt }) })

Step 4: Display Generated Image

Add the returned image URL to your canvas/editor

Quick Integration Option:

Use the Pre-built React Component

If you're using React, simply import and use:

import AIImageGenerator from './AIImageGenerator';
<AIImageGenerator isOpen={showAI} onClose={() => setShowAI(false)} onImageGenerated={(url) => addToCanvas(url)} />

See ai_service/INTEGRATION_GUIDE.md for complete instructions.
