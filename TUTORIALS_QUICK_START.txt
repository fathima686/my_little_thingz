=====================================================

cd frontend

Frontend will be available at: http://localhost:5173

=====================================================
STEP 3: TEST THE FLOW
=====================================================

1. Go to homepage: http://localhost:5173

2. Scroll down to "Learn the Art of Gift Making" section

3. Click "Study Tutorials" button
   - If not logged in → redirects to login
   - If logged in → goes to tutorials dashboard

4. On tutorials dashboard:
   - View 5 sample tutorials
   - See free tutorial: "Introduction to Photo Editing"
   - See paid tutorials with prices (₹99 - ₹299)

5. Try watching free tutorial:
   - Click "Watch Now" on free tutorial
   - Should redirect to tutorial viewer with video

6. Try purchasing paid tutorial:
   - Click "Purchase" on any paid tutorial
   - Payment modal opens
   - (Test with Razorpay test credentials if available)

DATABASE STRUCTURE
=====================================================

tutorials table:
- id, title, description, thumbnail_url
- video_url, duration, difficulty_level
- price, is_free, category
- created_by, created_at, updated_at, is_active

tutorial_purchases table:
- id, user_id, tutorial_id
- purchase_date, expiry_date
- payment_method, razorpay_order_id, razorpay_payment_id
- payment_status, amount_paid

=====================================================
TROUBLESHOOTING
=====================================================

Issue: "Study Tutorials" button not visible
→ Clear browser cache, restart dev server

Issue: Can't access tutorials without login
→ This is expected behavior (protected route)

Issue: Tutorial video not loading
→ Check video_url in database is valid embed URL
→ For YouTube use: https://www.youtube.com/embed/VIDEO_ID

Issue: Payment modal won't open
→ Check browser console for errors
→ Verify Razorpay config exists in backend

See TUTORIALS_SETUP.txt for more troubleshooting
