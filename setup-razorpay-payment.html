🚀 Razorpay Payment Setup

Complete guide to set up real Razorpay payments in your application

Step 1: Create Razorpay Account

If you don't have a Razorpay account yet:

  1. Go to Razorpay Signup
  2. Fill in your business details
  3. Verify your email and phone number
  4. Complete the KYC process (for live payments)
Sign Up for Razorpay

Step 2: Generate API Keys

Once you have access to the Razorpay Dashboard:

  1. Log in to Razorpay Dashboard
  2. Go to Settings → API Keys
  3. Make sure you're in Test Mode (for development)
  4. Click "Generate Key"
  5. Copy both the Key ID and Key Secret
Go to API Keys

Step 3: Update Configuration

Update your Razorpay keys in the configuration file:

Option A: Edit razorpay-keys.php (Recommended)

Open backend/config/razorpay-keys.php and replace:

'test' => [ 'key_id' => 'rzp_test_YOUR_ACTUAL_KEY_ID', 'key_secret' => 'YOUR_ACTUAL_SECRET_KEY' ],

Option B: Environment Variables

Create a .env file in your backend folder:

RAZORPAY_KEY_ID=rzp_test_your_key_id_here RAZORPAY_KEY_SECRET=your_secret_key_here

For all test cards:

  • CVV: Any 3 digits (e.g., 123)
  • Name: Any name

Step 4: Test Payment Flow

  1. Start your application (frontend and backend)
  2. Log in as a customer
  3. Add items to cart
  4. Fill shipping address
  5. Click "Pay Securely"
  6. Use test card details from above
  7. Complete the payment

⚠️ Important Notes

  • Test Mode: No real money is charged in test mode
  • API Keys: Test keys start with rzp_test_
  • Live Mode: Only switch to live mode after thorough testing
  • Security: Never share your secret keys publicly

Step 5: Going Live (Production)

When you're ready for real payments:

  1. Complete business verification in Razorpay Dashboard
  2. Switch to Live Mode in dashboard
  3. Generate Live API keys
  4. Update configuration with live keys
  5. Change mode to 'live' in razorpay-keys.php
  6. Test with small amounts first

✅ Troubleshooting

  • Authentication Failed: Check if API keys are correct and properly formatted
  • Invalid Key: Ensure you're using Test keys in Test mode
  • Network Error: Check if your server can reach Razorpay APIs
  • CORS Error: Ensure proper headers are set in backend APIs

🎉 Ready to Accept Payments!

Once configured, your customers can pay securely using cards, UPI, wallets, and more.
