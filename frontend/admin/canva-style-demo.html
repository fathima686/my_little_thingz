Integration Guide

1

Database Setup

Run the SQL schema to create template tables:

mysql -u username -p database_name < backend/sql/template-system-schema.sql
2

File Structure

Copy the new files to your project:

frontend/admin/canva-style-editor.html frontend/admin/js/canva-style-editor.js backend/api/admin/template-gallery.php
3

API Configuration

Update the API base URL in the JavaScript file:

// In canva-style-editor.js this.API_BASE = "http://your-domain.com/backend/api";
4

Integration with Existing Workflow

Modify the completeDesign() function to integrate with your existing order system:

async completeDesign() { const payload = { action: 'complete', order_id: this.currentOrderId, design_data: this.canvas.toJSON(), final_image_url: this.canvas.toDataURL() }; // Your existing API call const response = await fetch('/your-existing-endpoint', { method: 'POST', body: JSON.stringify(payload) }); }
5

Replace Existing Editor

Update your admin dashboard to use the new editor:

// Replace old editor link <a href="admin/canva-style-editor.html?order_id=123">Edit Design</a>
6

Optional Enhancements

Add background removal API integration:

// In removeImageBackground() function const response = await fetch('https://api.remove.bg/v1.0/removebg', { method: 'POST', headers: { 'X-Api-Key': 'YOUR_REMOVE_BG_API_KEY' }, body: formData });
