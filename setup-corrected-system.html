🔍 How the Corrected System Works

  1. Upload: Student uploads practice images
  2. Categorization: System identifies tutorial category (embroidery, painting, etc.)
  3. Hash Generation: Creates perceptual hash (pHash) of each image
  4. Comparison: Compares only against images in same category
  5. Similarity Check: Calculates Hamming distance (≤5 = similar)
  6. Decision: Clear status (unique/reused/highly_similar/needs_review)
  7. Admin Review: Flagged images go to admin for manual review
  8. Progress: Only admin-approved practice counts toward certificates
