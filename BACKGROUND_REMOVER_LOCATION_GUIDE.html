🎨 Background Remover Feature

Complete guide to using the background removal tool in your design editor

✅ Already Installed

📍 Where to Find It

The Remove Background button appears in the Image Properties Panel on the right sidebar when you select an image.

Button Location:

Right Sidebar → Image Properties Panel → Image Tools

🪄 Remove Background

🚀 How to Use (Step by Step)

1

Upload an Image

Click the "Add Image" button in the right sidebar (under "Add Elements" section) and select an image file from your computer.

2

Select the Image

Click on the uploaded image in the canvas to select it. You'll see selection handles around the image.

3

Open Image Properties

The right sidebar will automatically show "Image Properties Panel" with the image tools.

4

Click "Remove Background"

Click the purple gradient button labeled "Remove Background" with the magic wand icon (🪄).

5

Wait for Processing

The system will process your image (2-5 seconds). You'll see a loading indicator.

6

Done!

The background will be removed and the image will be updated on the canvas. You'll see a success message.

⚙️ Two Processing Methods

🌟 Method 1: API-Based (Professional)

  • High-quality background removal
  • Works on complex backgrounds
  • Requires remove.bg API key
  • Free tier: 50 images/month

Setup:

# Add to .env file: REMOVE_BG_API_KEY=your_api_key_here # Get free API key from: https://remove.bg/api

⚡ Method 2: Client-Side (Basic)

  • Basic background removal
  • Works on simple/white backgrounds
  • No API key needed
  • Unlimited use, completely free

Setup: None required - works automatically!

🎯 What Happens Automatically

The system is smart and will:

  1. Check for API Key - If configured, uses professional API
  2. Fallback to Client-Side - If no API key, uses basic processing
  3. Show Clear Messages - Tells you which method was used
  4. Preserve Image Properties - Keeps position, size, and rotation

💡 Tips for Best Results

✅ Do This

  • Use high-quality images
  • Keep file size under 5MB
  • Use API for complex backgrounds
  • Test with simple images first

❌ Avoid This

  • Very low resolution images
  • Images larger than 5MB
  • Expecting perfection without API
  • Complex backgrounds without API

🔧 Troubleshooting

Button not showing?

✓ Make sure you've selected an image (click on it)

✓ Check that the right sidebar is visible

✓ Refresh the page and try again

Not working well?

✓ For better results, add remove.bg API key

✓ Try with a simpler background first

✓ Check browser console for errors

API error?

✓ Verify API key is correct in .env file

✓ Check you haven't exceeded monthly quota

✓ System will automatically use client-side as fallback
