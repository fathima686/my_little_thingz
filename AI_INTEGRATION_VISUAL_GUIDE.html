🔧 Troubleshooting

❌ "Service Offline" message?

Solution: Start the AI service
cd ai_service
.\venv\Scripts\python.exe main.py

❌ Not seeing the button?

Solution: Hard refresh browser
Press: Ctrl + Shift + R
Or: Ctrl + F5

❌ "No routes matched" error?

This should NOT happen! If it does:
  • Clear browser cache completely
  • Hard refresh (Ctrl+Shift+R)
  • Restart React dev server

⏱️ Slow generation?

This is normal!
  • First generation: 2-5 minutes (model loading)
  • Subsequent: 30-90 seconds (CPU processing)
  • Be patient and don't refresh!
